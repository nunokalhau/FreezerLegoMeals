# 🥩 Proteins

This folder contains the main protein recipes used as building blocks in the meal-prep system.

## What belongs here

Use this folder for recipes such as chicken, beef, chili, tofu, and other protein-focused components.

## How to use it

Each recipe should be structured so it can be reused across different meals and combined with multiple bases and vegetables.

## Relationship to the rest of the project

Proteins are one of the core components of the system, alongside starches, vegetables, sauces, and pickles.
