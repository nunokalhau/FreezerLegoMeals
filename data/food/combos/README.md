# 🍽️ Combos

This folder contains example meal combinations built from the core components in the repository.

## What belongs here

Use this folder to show how proteins, vegetables, starches, sauces, and pickles can be combined into complete meals.

## How to use it

These examples are meant to make the system practical and reduce decision fatigue during the week.

## Relationship to the rest of the project

Combos connect the modular components into real meal ideas and show how the system works in practice.
