# 🥫 Sauces

This folder contains sauces and finishing flavors that add variety to the meal-prep system.

## What belongs here

Use this folder for sauces, creams, and other flavor boosters that can change the character of a meal without changing its core structure.

## How to use it

These recipes should be easy to portion and useful across multiple meal combinations.

## Relationship to the rest of the project

Sauces help connect proteins, starches, and vegetables into more complete and interesting meals.
