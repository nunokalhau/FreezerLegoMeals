# 🥦 Veggies

This folder contains vegetable recipes that complement the protein and base components in the meal-prep system.

## What belongs here

Use this folder for roasted vegetables, braised vegetables, mash-style sides, and other freezer-friendly vegetable components.

## How to use it

These recipes should be modular, easy to portion, and suitable for combining with different proteins and starches.

## Relationship to the rest of the project

Vegetables are a core building block that helps create balanced and varied meals.
