# 🥒 Pickles

This folder contains pickles and other preserved or flavor-enhancing additions that can brighten the meal-prep system.

## What belongs here

Use this folder for quick pickles, preserved toppings, and other small components that add acidity, crunch, or contrast.

## How to use it

These items should be modular and easy to pair with different proteins, starches, and vegetables.

## Relationship to the rest of the project

Pickles add variety and help turn simple meal blocks into more complete and interesting meals.
