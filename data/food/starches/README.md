# 🍚 Starches

This folder contains the base components that turn meal blocks into complete meals.

## What belongs here

Use this folder for rice, pasta, beans, and other starch-like bases that pair well with proteins and vegetables.

## How to use it

These recipes should be easy to reheat, simple to portion, and flexible enough to combine with different meal components.

## Relationship to the rest of the project

Starches are one of the core building blocks in the system and work together with proteins, vegetables, sauces, and pickles.
