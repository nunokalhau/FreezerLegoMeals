# Guides

This folder contains practical workflow guidance for using the repository as a meal-prep system.

## What belongs here

Use this folder for batch-cooking instructions, planning advice, shopping guidance, and other operational notes.

## How to use it

The content here should make the system easier to follow in practice and reduce friction when moving from recipes to actual meal prep.

## Relationship to the rest of the project

Guides connect the recipe content with real-world usage and help make the project easier to adopt.
