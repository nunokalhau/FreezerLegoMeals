# Contributing to Freezer Lego Meals

This repository works best when new content follows the same structure and intent.

## General guidelines

- keep recipes practical and freezer-friendly
- prefer simple, repeatable steps over complicated techniques
- use the category templates whenever you add a new recipe
- document freezing, reheating, and combinations clearly
- if a new category is added, update the relevant README files and navigation

## Recipe format

Each recipe should include:

- a clear title
- ingredients
- preparation steps
- freezing guidance
- reheating guidance
- suggested combinations
- short notes when helpful

## Documentation expectations

If you change the structure of the repository or introduce a new workflow, update:

- the relevant category README
- the main README
- the architecture notes in this folder
- the roadmap if the direction changes meaningfully

## Suggested workflow

1. choose the right category for the recipe
2. copy the matching template
3. fill in the recipe details
4. add a short description to the category README if needed
5. keep the content easy to scan and easy to reuse
